AI Text Detector - Детектор сгенерированного текста
Мультиплатформенный инструмент (Десктопное приложение + Веб-сайт) для быстрого анализа текстов на предмет генерации искусственным интеллектом. Система выполняет точную бинарную классификацию (Человек / ИИ) с выводом процентной вероятности.

🛠 Текущий функционал и особенности
Умная обработка файлов: Поддержка файлов размером до 1 ГБ (.txt и .pdf). Текст извлекается в фоне и обрезается до 50 000 символов, что защищает интерфейс от зависаний при чтении огромных книг.

Защита интерфейса: При загрузке файла реальный текст скрывается под системную заглушку, а само поле ввода временно блокируется.

Система предупреждений: Программа предупреждает пользователя, если анализируется слишком короткий текст (менее 100 символов), так как в этом случае точность классификации падает.

Десктоп на Qt: Полностью переписанный интерфейс на базе PySide6 с нативной поддержкой горячих клавиш (Ctrl+C, Ctrl+V, Ctrl+A) на любой раскладке клавиатуры.

Современный Веб-сайт: Облегченная веб-версия с поддержкой парсинга PDF на стороне клиента (через PDF.js) и стильным ночным (темным) интерфейсом.

📜 Полная история изменений (Changelog)
Оптимизация производительности и лимитов
Константы лимитов: Введены жесткие ограничения: минимальный порог информативности в 100 символов (с выводом предупреждения) и максимальный объем обрабатываемого текста в 50 000 символов.

Контроль памяти (1 ГБ): Логика ограничения файлов переведена с подсчета символов на проверку реального веса файла в байтах. Файлы крупнее 1 ГБ отсекаются еще до попытки чтения, предотвращая утечку оперативной памяти.

Глубокий рефакторинг UI и UX
Переход на Qt (PySide6): Устаревший и визуально ограниченный фреймворк Tkinter полностью заменен на PySide6. Это обеспечило современный внешний вид (тема Fusion) и стабильную работу с буфером обмена.

Фокус на главном: Из названия приложения удалена техническая пометка (Кейс 5). Интерфейс очищен от лишних элементов: удалена кнопка «Выделить всё», так как стандартные комбинации клавиш в Qt теперь работают идеально.

Отказ от классификации моделей: Из десктопной и веб-версии полностью удалена функция определения конкретной модели ИИ (убран чекбокс и упоминания в инструкциях). Проект сфокусирован на чистой бинарной проверке «ИИ или Человек».

Улучшение информативности результатов
Интеграция процентов: Процентная вероятность генерации теперь выводится не только в сохраняемый лог-файл, но и наглядно отображается в интерфейсе:

Крупным шрифтом прямо в строке финального вердикта: Похоже на: ИИ (87.5%).

Текстом прямо по центру цветной полосы прогресса (Progress Bar).

Модернизация веб-версии (index.html)
Клиентский парсинг PDF: Благодаря интеграции библиотеки PDF.js (Mozilla), сайт теперь умеет открывать и поклинейно читать файлы .pdf без использования серверной части.

Ночная тема (Dark Mode): Веб-интерфейс полностью перекрашен в приятные глубокие темные тона (#121214 и #1e1e24) для снижения нагрузки на глаза при долгой работе.

Удаление лишних метрик: Из блока результатов стерты отвлекающие математические метрики (ROC-AUC, F1-score), делая сервис максимально понятным для рядового пользователя.

🚀 Инструкция по запуску
1. Десктопное приложение (Python)
Требования: Python 3.8+

Установите необходимые зависимости:

Bash
pip install PySide6 pypdf
Запустите скрипт:

Bash
python main.py
2. Веб-версия (GitHub Pages / Локально)
Веб-версия является полностью автономной (Serverless).

Достаточно открыть файл index.html в любом современном браузере.

Для публикации просто загрузите файл index.html в ваш репозиторий GitHub и включите GitHub Pages в настройках. Библиотека для чтения PDF подключится автоматически через CDN.
